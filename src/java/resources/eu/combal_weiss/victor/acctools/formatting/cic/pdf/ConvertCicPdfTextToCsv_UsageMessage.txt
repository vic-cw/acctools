
Usage:  java {0}.{1} [-h]

        Read text representation of pdf from standard input, trim it to 
        keep only list of transactions, and print result to standard output
        in csv format.

        Accepts only text extracted using "pdftotext -table" from pdf files 
        generated by CIC.

Options:
        -h  Display this help message

Exit code:
        0   if everything goes well
        1   otherwise

EXAMPLE:
        cat transformed_pdf.txt | java {0}.{1} >trimmed_pdf.csv
